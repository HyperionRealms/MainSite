Lead Backend Programmer

Full Time, Remote

Overview

Hyperion Realms is seeking a Lead Backend Engineer with a passion for building scalable and reliable backend web services.

We are looking for someone with an innate curiosity to always dig deeper, to follow the latest trends in their field of expertise and constantly push the solutions further.

This role will start out being very hands-on directly implementing services, but will later evolve into a mixed management role of leading a small team of backend developers.

You will join a small closely-knit team of ambitious game makers working on defining the next evolution of the survival games genre, and as an early employee you’ll have the ability to influence the direction of both the game and how we build our culture.

Who We Are

Hyperion Realms is a new video game development startup creating an exciting survival and exploration game for PC and consoles. We are a fully remote company and welcome diverse team members from around the world. Learn more about us at https://hyperionrealms.com.

Responsibilities

  • Setup cloud hosting infrastructure to run secure game persistence backend services, and game hosting in the cloud
  • Write high-performance C# code using the latest versions of .NET (v6 or 7), using either GRPC or Web API frameworks
  • Write highly scalable cloud-based solutions that can easily scale capacity on demand
  • Setup central authentication and authorization for all services, work with gameplay team on gameplay client interfacing with authentication
  • Work with game design on user access strategies for content
  • Work with gameplay engineers on anti-cheating measures, mainly to track and validate what user has access to
  • Design and write database storage for backend services
  • Write and maintain build- and deployment processes for backend services
  • Collaborate with gameplay team on game server deployment and management
  • Write and perform load testing on backend services
  • Write high quality maintainable code
  • Discover and fix bugs in existing systems

Qualifications

  • 5 years of experience writing backend services in the .NET tech stack
  • Expert C# skills
  • Experience writing backend services using GRPC or Web API
  • Experience developing database solutions using relational SQL databases
  • Experience using caching technologies to accelerate response time of backend services
  • Good knowledge of cloud-based hosting and deployment services using containers
  • Experience setting up and running automated build- and deployment services
  • Experience running Continuous Integration loops to constantly test and deploy latest code changes
  • At least basic knowledge managing Linux and Windows servers
  • Excellent written and verbal communication in English
  • A proactive mindset focussed on building great experiences for players
  • An eagerness to embrace new tech and solve old problems in new ways

Advantages

  • Experience working on a live service game
  • Experience with Unreal Engine

How To Apply

Send your resume and cover letter to jobs@hyperionrealms.com and include the job title in the subject of the email. We'll review your info and get back to you promptly.

Technical Artist

Full Time, Remote

Overview

Hyperion Realms is seeking a Technical Artist with a diverse skill set to play an all-rounder role as we enter our early development phases.

This is a critical role which will tackle a variety of challenges from optimization to bug fixing to pipeline development, all using the latest Unreal Engine technology. Working with other artists and engineers you will have the chance to drive the visual development, direction, and vision of the game.

You will join a small closely-knit team of ambitious game makers working on defining the next evolution of the survival games genre, and as an early employee you’ll have the ability to influence the direction of both the game and how we build our culture.

Who We Are

Hyperion Realms is a new video game development startup creating an exciting survival and exploration game for PC and consoles. We are a fully remote company and welcome diverse team members from around the world. Learn more about us at https://hyperionrealms.com.

Responsibilities

  • Establish and maintain art pipelines that let artists focus on creative work
  • Optimize game performance in a huge realistic procedurally generated world
  • Technically prepare assets for use in game with rigging and material setup
  • Master the latest UE5 features and teach the team to make the most of them
  • Work with design and engineering to generate procedural worlds at runtime
  • With the art director, find an aesthetic that takes advantage of the latest tech

Qualifications

  • At least 2 years experience as a technical artist working in Unreal Engine
  • Expertise with materials, blueprints, and all aspects of the UE art pipeline
  • Excellent english language skills (english is our common language as a team)
  • A “can do” attitude focussed on solving problems in new and creative ways
  • An outlook that embraces both the latest features and traditional know-how
  • Be willing to wear multiple hats and support the team in a variety of ways

Advantages

  • Expertise in technical art, rigging, animation, VFX, or other specialties
  • Experience working with the latest features offered by Unreal Engine 5
  • Experience using and/or authoring procedural generation systems

How To Apply
